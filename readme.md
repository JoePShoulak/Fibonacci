# Fibonacci #

## About ##
Lists all the Fibonacci numbers up to and including the given index, or simply the Fibonacci number of the given index, if including the `-e` flag.

## Usage ##
    $ fibonacci 5
      // 1: 1
      // 2: 1
      // 3: 2
      // 4: 3
      // 5: 5
    $ fibonacci 10 -e
      // 10: 55
